### CSS Code:


```css
        <!-- Custom css -->
        <link rel="stylesheet" href="css/style.css">
        <!-- Font awesome 5 css -->
        <link rel="stylesheet" href="css/fontawesome.min.css">
        <!--Owl Carousel css -->
        <link rel="stylesheet" href="css/owl.carousel.css">
        
```

### JS Code:
```css
  <!-- ALL JS HERE-->
```
